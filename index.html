Commandes Git

Git init

Initialiser un nouveau repo

                        git init "Nom du repo git"

Git remote

Lier un dépôt local au dépôt distant

                        git remote add origin https://github.com/account/project.git

Git clone

Cloner un dépôt déjà existant

                        git clone https://github.com/account/project.git

Git add

Ajouter un fichier en zone staging

                        git add index.html

Ajouter tous les fichiers en zone staging

                        git add *

Git commit

Faire un commit de son code avec un message

                        git commit -m "le message"

Git branch

Liste toutes les branches du dépôt

                        git branch

Créer une nouvelle branche sur le dépôt

                        git branch "nom de la branche"

Supprimer une branche

                        git branch -d "nom de la branche"

Git checkout

Pour passer d'une branche à l'autre

                        git checkout

Créer une nouvelle branche et switcher directement sur celle-ci

                        git checkout -b "nom de la branche"

Git push

Pour envoyer les commits sur le dépôt distant

                        git push -u origin main
